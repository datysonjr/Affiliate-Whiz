## The TRUE Affiliate Site Shape (not a blog)

Bad sites look like:

```
Home
 ├ blog post
 ├ blog post
 ├ blog post
```

These almost never scale.

Real money sites look like this:

```
HOME
 │
 ├── HUB PAGE (Main Buyer Intent Topic)
 │     ├── MONEY PAGE (Best Product / Service)
 │     ├── MONEY PAGE (Comparison)
 │     ├── BUYING GUIDE
 │
 ├── SUPPORT CLUSTER (Informational)
 │     ├── how to choose X
 │     ├── mistakes with X
 │     ├── X vs Y
 │     ├── beginner guide to X
 │
 ├── REVIEW PAGES
 │     ├── product review A
 │     ├── product review B
 │     ├── product review C
```

This is called a **Hub → Cluster → Conversion** topology.

OpenClaw should generate sites in this pattern automatically.

## The 5 Page Types That Actually Make Money

OpenClaw should ONLY generate these types. No random blog fluff.

### 1) HUB PAGE (Authority Anchor)

Examples:
- Best Electric Standing Desks
- Best Credit Card Processors
- Best Budget Studio Monitors

Purpose:
- rank for main keyword
- central authority page
- link target for every other article

Characteristics:
- longform
- structured sections
- internal links everywhere
- comparison tables
- buyer-intent focused

This is the traffic magnet.

### 2) MONEY PAGE (Highest conversion)

Examples:
- Top 10 Standing Desks Under $500
- Best Payroll Software for Small Business
- Best DSLR Camera for Beginners

Purpose:
- maximum affiliate clicks
- direct purchase intent

These convert the best.

### 3) BUYING GUIDE (Trust builder)

Examples:
- How to Choose a Standing Desk
- What To Look For In Payroll Software
- Studio Monitor Buying Guide

Purpose:
- informational intent
- pre-purchase education
- funnels users toward money pages

These help Google trust the site.

### 4) COMPARISON PAGES (Super high conversion)

Examples:
- Uplift vs Fully Desk
- QuickBooks vs Gusto
- Sony A7III vs Canon R6

These convert insanely well because user intent = decision stage.

OpenClaw should prioritize these heavily.

### 5) MICRO SUPPORT ARTICLES

Examples:
- Is a standing desk worth it?
- How tall should a standing desk be?
- Are standing desks good for back pain?

Purpose:
- capture long-tail search traffic
- build topical authority
- link toward hub + money pages

These make ranking easier.

## The Secret: Internal Linking Flow

Money sites don't rely on Google alone.
They build their own conversion funnel via links.

Correct flow:

```
SUPPORT ARTICLE
       ↓
  BUYING GUIDE
       ↓
   MONEY PAGE
       ↓
 AFFILIATE LINK
```

OpenClaw should generate internal links like this automatically.
Never random linking.

OpenClaw should include these automatically.

These boost conversions dramatically.

Never just text links.

## Page Type to Agent Mapping

| Page Type | Content Agent Template | Priority | Internal Links To |
|---|---|---|---|
| Hub Page | `hub_authority` | 1 (build first) | Money pages, buying guides |
| Money Page | `product_review`, `roundup` | 2 | Hub page, comparison pages |
| Comparison Page | `comparison` | 2 | Hub page, money pages |
| Buying Guide | `how_to` | 3 | Hub page, money pages |
| Support Article | `informational` | 4 | Hub page, buying guides |

### Link Direction Rules

```
Support → Buying Guide → Money Page → Affiliate Link
   ↓            ↓              ↓
  Hub ←←←←←←← Hub ←←←←←←←← Hub
```

Every page links back to the hub.
Support flows upward toward conversion.
Hub accumulates all authority.

## Site Build Order (Automated)

OpenClaw should create pages for a new site in this exact order:

1. Hub page (1 per niche)
2. Top 3 money pages (best-of, top-10, roundup)
3. Top 2 comparison pages (vs articles)
4. 1 buying guide
5. 3-5 support articles

Only then: evaluate performance and expand winning clusters.

This prevents the "spray and pray" failure mode.
