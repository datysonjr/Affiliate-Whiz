# Weekly Audit Checklist

## Performance
- [ ] Review revenue by site (compare to previous week)
- [ ] Review traffic by site (compare to previous week)
- [ ] Check top 10 performing pages
- [ ] Check bottom 10 performing pages (prune candidates)
- [ ] Review LLM API costs for the week

## Content
- [ ] Count articles published this week
- [ ] Spot-check 5 articles for quality
- [ ] Review content in error/quarantine state
- [ ] Check content queue depth for next week

## SEO
- [ ] Review Google Search Console for new issues
- [ ] Check indexing status of recently published pages
- [ ] Review keyword ranking changes
- [ ] Check for broken links (internal and external)

## System Health
- [ ] Review error rates and error types
- [ ] Check disk space on both nodes
- [ ] Review agent performance metrics
- [ ] Check backup success/failure log

## Security
- [ ] Review access logs for anomalies
- [ ] Check for any credential expiration warnings
- [ ] Verify vault is accessible and healthy

## Action Items
- [ ] Document issues found
- [ ] Create tickets for fixes
- [ ] Update thresholds if needed
- [ ] Plan next week's priorities
