# Monthly Audit Checklist

## Revenue & ROI
- [ ] Total revenue across all sites
- [ ] Revenue per site breakdown
- [ ] Revenue per niche breakdown
- [ ] Total costs (hosting, LLM API, tools, domains)
- [ ] Net profit calculation
- [ ] ROI per site
- [ ] Compare to previous month and 3-month trend

## Content Portfolio
- [ ] Total articles across all sites
- [ ] Articles published this month
- [ ] Content quality score distribution
- [ ] Prune candidates identified and actioned
- [ ] Top performing content analysis (what's working)
- [ ] Content gap analysis (what's missing)

## SEO Health
- [ ] Overall organic traffic trend
- [ ] New keywords ranking (top 10, top 50)
- [ ] Keywords lost or declining
- [ ] Domain Authority changes
- [ ] Backlink profile review
- [ ] Core Web Vitals across all sites
- [ ] Crawl budget utilization

## Affiliate Programs
- [ ] Review commission rate changes
- [ ] Check for discontinued programs/offers
- [ ] Evaluate new affiliate programs to join
- [ ] Review network compliance status
- [ ] Update offer scores based on actual performance

## Infrastructure
- [ ] Hardware health check (both Mac Minis)
- [ ] Storage utilization trend
- [ ] Backup integrity verification (test restore)
- [ ] Security patches applied
- [ ] API key rotation (if due)
- [ ] SSL certificate expiry check

## Strategic Planning
- [ ] Niche performance review (expand/contract)
- [ ] New site opportunities
- [ ] Tool and provider evaluation
- [ ] Process improvement ideas
- [ ] Update PLAN.md with next month's priorities
