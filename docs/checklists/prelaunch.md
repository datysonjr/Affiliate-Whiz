# Pre-Launch Checklist

## Site Setup
- [ ] Domain registered and DNS configured
- [ ] SSL certificate active
- [ ] CMS installed and configured
- [ ] Theme/template applied and customized
- [ ] Favicon and branding assets uploaded
- [ ] robots.txt configured
- [ ] sitemap.xml generating correctly

## Legal & Compliance
- [ ] Privacy policy page published
- [ ] Terms of service page published
- [ ] Affiliate disclosure page published
- [ ] Cookie consent banner active
- [ ] FTC disclosure in footer/header

## Content
- [ ] Minimum 10 posts published before launch
- [ ] About page published
- [ ] Contact page published
- [ ] Categories and tags organized
- [ ] Internal links verified
- [ ] All affiliate links tested and tracking

## Technical
- [ ] Page speed score >80 (mobile)
- [ ] Core Web Vitals passing
- [ ] Mobile responsive verified
- [ ] Schema markup validated
- [ ] 404 page customized
- [ ] Redirects configured

## Analytics
- [ ] Google Analytics installed
- [ ] Google Search Console verified
- [ ] Affiliate link click tracking active
- [ ] Conversion tracking configured

## SEO
- [ ] Title tags and meta descriptions set for all pages
- [ ] H1 tags present and correct
- [ ] Image alt text filled in
- [ ] Internal linking structure in place
- [ ] Sitemap submitted to Google Search Console
