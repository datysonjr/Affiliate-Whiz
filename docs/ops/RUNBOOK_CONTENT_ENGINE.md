# RUNBOOK_CONTENT_ENGINE.md — Article Generation Pipeline

## Inputs
- niche brief
- target keyword(s)
- search intent (informational vs commercial)
- internal linking targets
- affiliate products list (compliant)

## Outputs
- outline
- draft markdown
- SEO metadata (title/meta/slug)
- suggested internal links
- optional schema placeholders

## Quality Checklist
- Helpful, original, readable
- Not keyword-stuffed
- Includes pros/cons where appropriate
- Includes disclosure
- Clear CTA without deception

## Review Gates
- Optional human review step before production publish
- Staging publish always allowed for testing
