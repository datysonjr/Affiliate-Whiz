# RUNBOOK_SEO_INTERNAL_LINKING.md — Structure & Linking

## Goals
- Build topical authority
- Improve crawlability
- Improve engagement

## Internal Linking Logic
- Each post links to:
  - 1 hub page
  - 2-4 related posts
  - 1 "best of" / money page (if relevant)

## Avoid
- Over-optimized anchor spam
- Link farms
- Deceptive doorway pages

## Site Map
Maintain an internal map:
- cluster topics
- hubs
- supporting articles
- money pages
