# RUNBOOK_SITE_FACTORY.md — Website Deployment Template

## Goal
Deploy a new niche site with consistent structure and compliance.

## Standard Site Structure
- Homepage (niche overview)
- Category pages (topic clusters)
- About page (who we are + disclosure)
- Contact page
- Privacy policy
- Affiliate disclosure page

## Technical Standards
- Fast theme, minimal bloat
- Clean permalinks
- Caching enabled
- Image optimization
- XML sitemap enabled

## Content Standards
- No fake claims
- No plagiarized blocks
- Real product comparisons where possible
- Clear affiliate disclosure

## Publishing Gate
- Staging first
- Production requires explicit approval
