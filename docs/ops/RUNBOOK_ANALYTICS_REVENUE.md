# RUNBOOK_ANALYTICS_REVENUE.md — Performance & Revenue Monitoring

## Minimum Metrics
- number of posts published
- impressions/clicks (Search Console)
- sessions (analytics)
- affiliate clicks (if tracked)
- conversions/revenue (affiliate dashboard when available)

## Cadence
- Daily: health + failures
- Weekly: traffic trend + top pages
- Monthly: niche ROI review + scale decisions

## Alerts
- publishing failures
- traffic drop beyond threshold
- indexation failures
- repeated job retries
