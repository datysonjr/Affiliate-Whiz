# OPENCLAW_90_DAY_RAMP_PLAN.md

OpenClaw Affiliate Automation System

Purpose: Defines the realistic 90-day launch progression for a new OpenClaw site or cluster.

This plan ensures:

- safe publishing cadence
- correct SEO signal interpretation
- stable authority growth
- controlled automation scaling
- prevention of premature expectations

This document must be consulted during the first 3 months of any deployment.

---

## Core Truth

SEO affiliate systems do NOT produce money immediately.

They produce signals first.

Correct ramp sequence:

```
crawl → index → impressions → ranking movement → traffic → clicks → revenue
```

Revenue is the final stage, not the first.

---

## DAYS 1–7 — INFRA STABILITY PHASE

Goal: **Prove pipeline works safely.**

Publishing:

```
1 article per day MAX
```

Focus:

- CMS stability
- formatting correctness
- sitemap functionality
- uptime monitoring

Expected signals:

- pages appear in sitemap
- pages crawlable
- indexing may begin

DO NOT EXPECT:

- traffic
- rankings
- impressions

If publishing errors occur: **STOP and fix immediately.**

---

## DAYS 8–14 — INDEXING CONFIRMATION PHASE

Goal: **Confirm Google is recognizing site.**

Publishing:

```
3–5 articles per week
```

Focus:

- indexing rate
- crawl activity
- Search Console signals

Healthy signs:

- pages indexed within 3–7 days
- sitemap crawled regularly

Still DO NOT expect traffic.

---

## DAYS 15–30 — FIRST IMPRESSION PHASE

Goal: **Verify Google is testing content.**

Publishing:

```
4–6 articles per week
```

Focus:

- impressions starting to appear
- long-tail queries visible
- ranking entries in positions 30–70

**THIS IS THE FIRST REAL SUCCESS SIGNAL.**

Even:

```
20 impressions/day
```

means the system is working.

---

## DAYS 31–45 — RANKING MOVEMENT PHASE

Goal: **Observe position improvement.**

Publishing:

```
5–7 articles per week
```

Focus:

- pages moving toward top 30
- internal linking effectiveness
- cluster strengthening

Healthy signals:

- some pages reach position <30
- impression curve rising steadily

Still normal if clicks remain small.

---

## DAYS 46–60 — EARLY TRAFFIC PHASE

Goal: **First organic visitors.**

Publishing:

```
maintain 5–7 per week
```

Focus:

- click activity appearing
- session duration improving
- affiliate link clicks beginning

Healthy signals:

- 5–20 daily organic visits
- occasional affiliate click events

DO NOT scale aggressively yet.

---

## DAYS 61–75 — TRACTION PHASE

Goal: **Establish real ranking authority.**

Publishing:

```
maintain steady cadence
```

Focus:

- multiple pages ranking <20
- consistent impression growth
- early revenue signals possible

Healthy signs:

- 20–50 daily visits
- first measurable affiliate conversions

This is where momentum becomes visible.

---

## DAYS 76–90 — FIRST SCALE DECISION PHASE

Goal: **Determine if site ready for expansion.**

Evaluate:

- indexing stable?
- impressions rising weekly?
- ranking movement ongoing?
- affiliate clicks measurable?

If YES:

Allowed:

- expand cluster depth
- increase publishing cadence
- prepare second site launch

If NO:

Do NOT panic.

Investigate:

- keyword targeting
- cluster completeness
- internal linking strength
- competitor strength

**Never mass-publish to fix slow growth.**

---

## Healthy 90-Day Metrics (Typical)

Approximate healthy ranges:

```
Articles published:  60–120
Indexed pages:       70–90%
Daily impressions:   200–1500
Daily visits:        10–100
Affiliate clicks:    occasional to steady
Revenue:             possible but not guaranteed yet
```

Sites exceeding this are strong performers.

---

## Common Mistakes to Avoid

Never:

- publish 50 articles week one
- switch niches mid-ramp
- abandon site before day 90
- remove articles prematurely
- rebuild structure repeatedly

**Consistency beats speed.**

---

## Final Ramp Law

During the first 90 days:

Your job is NOT to chase money.

**Your job is to confirm signals.**

If signals exist, money becomes predictable.
