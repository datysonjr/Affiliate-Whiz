# Niche Selection Playbook

## Criteria for Niche Evaluation

### 1. Market Size & Demand
- Monthly search volume for primary keywords (>1,000/mo minimum)
- Trend direction (Google Trends - stable or growing)
- Seasonal patterns (avoid heavily seasonal unless diversified)

### 2. Competition Analysis
- Domain Authority of top 10 SERP results
- Content quality of existing top results
- Gaps in existing content (topics not well covered)
- Number of affiliate sites already ranking

### 3. Monetization Potential
- Average commission rate (>5% preferred)
- Average order value (>$50 preferred)
- Cookie duration (>30 days preferred)
- Number of available affiliate programs
- EPC (Earnings Per Click) data from networks

### 4. Content Feasibility
- Can we generate quality content at scale?
- Are there enough sub-topics for 50+ articles?
- Is the niche suitable for review/comparison content?
- YMYL considerations (avoid health/finance unless expert-backed)

## Scoring Matrix

| Factor | Weight | Score (1-5) |
|--------|--------|-------------|
| Search Volume | 20% | |
| Competition | 25% | |
| Commission Rate | 20% | |
| Order Value | 15% | |
| Content Scalability | 20% | |

**Minimum score to proceed: 3.5 / 5.0**

## Process

1. Generate niche candidates from trend data and network offers
2. Score each niche using the matrix above
3. Select top 3 niches for pilot sites
4. Create 10 articles per niche as initial test
5. Measure indexing, ranking, and click-through after 30 days
6. Double down on winners, pause losers

## Red Flags (Auto-reject)
- YMYL niches without expert review capability
- Niches with single dominant affiliate program (risk)
- Heavily regulated products (supplements, financial products)
- Niches requiring hands-on product testing we can't do
