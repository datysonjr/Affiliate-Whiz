# OpenClaw Programmatic SEO Rules

Programmatic SEO is REQUIRED.

But must follow safety rules.

---

## Allowed

- templated comparison pages
- templated "Best X for Y"
- templated review pages

---

## Forbidden

- auto-generated junk lists
- duplicate rewritten pages
- thin content

---

## Minimum Content Blocks

Each programmatic page must include:

- TLDR
- comparison table
- unique verdict paragraph
- FAQ
- internal links

If any missing → block publishing.
