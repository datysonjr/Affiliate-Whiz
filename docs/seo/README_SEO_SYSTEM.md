# OpenClaw SEO System

OpenClaw is designed to rank in:

1) Google search
2) Bing search
3) AI answer engines (ChatGPT / Claude / Perplexity / Bing Copilot)
4) Voice assistants
5) Featured snippets

This requires a different architecture than traditional affiliate SEO.

---

## Core Rule

OpenClaw pages must be:

- scannable
- extractable
- structured
- citation-friendly
- AI-readable

If a page is NOT extractable by an LLM in <5 seconds, it fails.

---

## Required Page Blocks (mandatory)

1. TLDR block (top of page)
2. comparison table
3. structured headings
4. FAQ block
5. internal links
6. product verdict section

NO page publishes without these.

---

## The Goal

Traditional SEO = rank for humans.

OpenClaw SEO = become the SOURCE AI cites.

That is the difference.
