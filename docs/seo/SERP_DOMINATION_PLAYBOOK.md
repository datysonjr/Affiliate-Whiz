# SERP_DOMINATION_PLAYBOOK.md

OpenClaw Affiliate Automation System

Purpose: Defines strategy for occupying multiple top-ranking positions for the same keyword ecosystem.

Goal: not just ranking once — but **owning the entire results page.**

---

## Core Principle

Google often shows:

- buyer guide
- comparison page
- informational page

for the SAME query ecosystem.

If we create all of them...

**we can occupy multiple positions.**

---

## Multi-Intent Page Strategy

For each primary keyword:

Create THREE page types:

### Buyer Guide Page

Example:

```
Best ergonomic office chairs
```

This is main revenue page.

### Comparison Page

Example:

```
Office chair vs gaming chair
```

Ranks for decision queries.

### Informational Guide

Example:

```
How to choose an office chair for back pain
```

Ranks for research queries.

---

## Internal Linking Loop

Each of the 3 page types must:

- link to each other
- reinforce topical authority
- share internal authority signals

This creates: **SERP cluster dominance.**

---

## Supporting Page Expansion

For each primary topic:

Create:

```
budget guide
premium guide
scenario-specific guides
mistake guides
alternatives page
```

These feed authority upward.

---

## Navigation Reinforcement

Top pages must:

- appear in category navigation
- appear in homepage feature block
- receive sidebar links

This signals site importance.

---

## Result

Instead of:

```
1 result in top 10
```

Goal becomes:

```
3–5 results in top 10 ecosystem
```

This multiplies traffic without new niches.

---

## Final Rule

Winning one keyword is good.

**Owning the entire query ecosystem is unstoppable.**
