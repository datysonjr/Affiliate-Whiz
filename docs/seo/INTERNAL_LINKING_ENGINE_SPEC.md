# INTERNAL_LINKING_ENGINE_SPEC.md

OpenClaw Affiliate Automation System

Purpose: Defines the automated internal linking strategy that drives ranking acceleration.

Internal linking is NOT optional.
It is the primary SEO leverage mechanism for affiliate sites.

---

## Core Principle

Google ranks: **CONNECTED TOPICS** not isolated pages.

A site with 30 strongly linked pages can outrank a site with 300 disconnected pages.

---

## Required Link Types

Each article MUST contain:

### 1. Parent Links

Link upward to:

- main category page
- main buyer guide

### 2. Sibling Links

Link to:

- related comparison pages
- alternative product guides

### 3. Child Links

Link downward to:

- detailed subtopics
- informational expansions

---

## Minimum Link Counts

Each article must have:

```
3 internal links minimum
6–10 preferred
```

---

## Anchor Text Strategy

Never:

```
click here
read more
this article
```

Always:

```
descriptive keyword-based anchors
```

Example:

```
See our guide to the best office chairs for small spaces.
```

---

## Link Timing Rule

When a new article publishes:

OpenClaw must:

1. update existing related pages
2. insert backlinks to new article
3. rebuild sitemap

This prevents orphan pages.

---

## Cluster Linking Pattern

Structure:

```
        MAIN GUIDE
            ↕
    SUPPORT ARTICLES
            ↕
  SUB-SUPPORT ARTICLES
```

Every node links:

- upward
- sideways
- downward

This creates authority loops.

---

## Revenue Reinforcement Links

High-conversion pages must receive:

- 2x more internal links
- homepage link priority
- navigation inclusion

This pushes money pages upward faster.

---

## Final Rule

Pages without internal links are invisible.

If OpenClaw detects:

```
internal_links < 3
```

**Publishing must be blocked.**
