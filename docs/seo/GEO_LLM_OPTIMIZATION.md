# OpenClaw LLM Optimization

Goal: Make OpenClaw pages chosen as AI answers.

---

## LLM Selection Behavior

AI chooses pages that are:

- clearly structured
- authoritative sounding
- contain explicit verdict statements
- contain tables
- contain FAQ blocks
- contain "Best X for Y" language

---

## Required Signals

Each article MUST include:

1. verdict sentences
2. explicit recommendations
3. comparison tables
4. short factual paragraphs
5. structured headings

---

## Forbidden Style

Do NOT publish:

- essay-style articles
- storytelling intros
- vague discussions
- opinion-heavy content

---

## Winning Pattern

AI prefers:

FACT → VERDICT → TABLE → FAQ

NOT:

INTRO → STORY → DISCUSSION → CONCLUSION
