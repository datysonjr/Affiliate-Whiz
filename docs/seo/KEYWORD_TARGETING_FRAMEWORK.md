# KEYWORD_TARGETING_FRAMEWORK.md

OpenClaw Affiliate Automation System

Purpose: Defines how OpenClaw selects WHICH articles to generate.

Wrong keyword targeting = infinite content with zero revenue.
Correct keyword targeting = exponential compounding traffic.

This framework ensures OpenClaw always prioritizes:

- achievable ranking opportunities
- commercial intent
- scalable topical clusters

---

## Core Philosophy

We do NOT chase:

- highest search volume
- trendy keywords
- generic head terms

We chase:

**LOW competition + CLEAR buying intent + scalable cluster potential**

---

## Keyword Qualification Rules

Every keyword must pass ALL three:

### 1. Intent Check

Must match one of:

```
best X for Y
X comparison
X review
X alternatives
X buyer guide
how to choose X
```

Reject:

- definition-only searches
- history searches
- news searches
- academic topics

### 2. Competition Reality Check

OpenClaw must reject keywords if:

- dominated by major brands
- dominated by government sites
- dominated by massive editorial publishers

Accept keywords if:

- results include small blogs
- results include niche sites
- results include thin affiliate pages
- outdated content exists

### 3. Monetization Linkability

Keyword must allow:

- at least 3 viable products
- affiliate links naturally included
- real buyer decisions involved

If no natural purchase exists → reject.

---

## Keyword Priority Scoring

Each candidate keyword receives:

```
Commercial Intent Score     (1–5)
Competition Weakness Score  (1–5)
Cluster Expansion Score     (1–5)
```

Total:

```
PRIORITY = Intent + Weakness + Cluster
```

Publish only if:

```
PRIORITY >= 10
```

---

## Cluster Expansion Logic

Each accepted keyword must support:

```
MAIN PAGE + 5–20 SUPPORTING PAGES
```

Example:

```
best ergonomic office chair
  → best chair for back pain
  → best office chair under $300
  → chair vs gaming chair
  → how to adjust office chair
```

If no cluster exists → reject keyword.

---

## Publishing Strategy

OpenClaw must:

1. publish MAIN page first
2. publish 3–5 support pages next
3. interlink immediately
4. expand cluster weekly

Never publish random isolated articles.
Isolated pages rarely rank. Clusters rank.

---

## Final Rule

If a keyword does not build topical authority...

**It does not get published.**
